# Game Backlog

A simple web app to track your video game backlog — what you own, what you're playing, what you've finished, and what's still waiting.

## About

Game Backlog lets you register games and organize them by status (*playing*, *finished*, *on hold*, *dropped*), along with details like platform, rating and hours played. The goal is to stop losing track of that ever-growing pile of unplayed games.

This is a learning project, built step by step as a hands-on way to practice Python, databases and web development end to end — from local setup to a deployed, working application.

## Tech Stack

- **Python** – core language
- **Flask** – web framework
- **SQLite + SQLAlchemy** – database and ORM
- **Jinja2** – HTML templating
- **pytest** – automated tests
